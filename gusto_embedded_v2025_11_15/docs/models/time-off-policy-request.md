# TimeOffPolicyRequest

Request body for creating a time off policy

## Example Usage

```typescript
import { TimeOffPolicyRequest } from "openapi/models";

let value: TimeOffPolicyRequest = {
  name: "Vacation Policy",
  policyType: "sick",
  accrualMethod: "per_anniversary_year",
};
```
