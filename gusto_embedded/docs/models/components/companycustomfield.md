# CompanyCustomField

A custom field on a company

## Example Usage

```typescript
import { CompanyCustomField } from "@gusto/embedded-api/models/components/companycustomfield.js";

let value: CompanyCustomField = {
  uuid: "73769bbc-a26a-439e-a167-9eb29246330c",
  name: "<value>",
  type: "radio",
};
```
