# CatchUpType

The type of catch-up contribution for this benefit, as required by Section 603 of the SECURE 2.0 Act. Only applicable to pre-tax 401(k) and 403(b) benefits.

## Example Usage

```typescript
import { CatchUpType } from "@gusto/embedded-api/models/components/companybenefit.js";

let value: CatchUpType = "deemed";
```

## Values

```typescript
"elective" | "deemed"
```
