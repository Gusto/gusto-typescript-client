# EmployeeFederalTaxRev2020

## Example Usage

```typescript

let value: EmployeeFederalTaxRev2020 = {
  version: "<value>",
  w4DataType: "rev_2020_w4",
  filingStatus: "<value>",
  extraWithholding: "<value>",
  twoJobs: false,
  dependentsAmount: "<value>",
  otherIncome: "<value>",
  deductions: "<value>",
};
```
