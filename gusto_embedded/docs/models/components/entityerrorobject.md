# EntityErrorObject

Will only exist if category is `nested_errors`. It is possible to have multiple levels of nested errors.

## Example Usage

```typescript
import { EntityErrorObject } from "@gusto/embedded-api/models/components";

let value: EntityErrorObject = {
  errorKey: "<value>",
  category: "<value>",
};
```
