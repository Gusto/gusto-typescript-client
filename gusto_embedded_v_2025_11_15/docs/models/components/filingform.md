# FilingForm

The form used by the company for federal tax filing. One of:
- 941 (Quarterly federal tax return form)
- 944 (Annual federal tax return form)

## Example Usage

```typescript
import { FilingForm } from "@gusto/embedded-api-v-2025-11-15/models/components/federaltaxdetails.js";

let value: FilingForm = "941";
```

```typescript
"941" | "944"
```
